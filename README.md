# Mobile-Based Market Linkage Platform for Smallholder Farmers in Ethiopia

## Overview

This project is a mobile-based market linkage platform designed to connect smallholder farmers, buyers, and transport service providers in Ethiopia. The platform aims to improve market transparency, reduce dependency on intermediaries, and facilitate direct transactions between stakeholders.

The system enables farmers to list agricultural products, buyers to discover and negotiate with sellers, and transporters to provide logistics services through a unified digital platform.

## Features

- User registration and authentication
- Role-based access (Farmer, Buyer, Transporter, Admin)
- Agricultural product listing and management
- Product search and filtering
- Buyer–farmer communication and negotiation
- Transport request and coordination
- Market information and price visibility
- Administrative management dashboard
- Multi-language ready architecture

## Objectives

- Improve market access for smallholder farmers
- Reduce information asymmetry in agricultural markets
- Enable direct farmer-to-buyer interactions
- Support efficient transport coordination
- Increase transparency and income opportunities within the agricultural value chain
